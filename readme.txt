Drag and drop file upload for ACF-Fields.

== Description ==

Enable Drag and Drop Upload in [ACF](https://www.advancedcustomfields.com/)â€™s Image, File and Gallery Field.

## Supported Fields

- Image Field
- File Field
- gallery Field

## Usage
Check "Enable Dropzone" in the ACF Field Editor.

## Development
Head over to the source code [on Github](https://github.com/mcguffin/acf-dropzone).
